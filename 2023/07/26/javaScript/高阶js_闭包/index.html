实质：闭包 本质当是作用域的问题，一个作用域有另一个作用域的访问权限就造就了前提条件，当有引用存在的时候就成闭包了，如果不能释放就 导致一直存储着

变量作用域

  • 变量作用域
    • 变量根据作用域的不同分为两种： 全局变量和局部变量
  1. 函数内部可以使用全局变量
  2. 函数外部不可以使用局部变量
  3. 当函数执行完毕，本作用域内的局部变量会销毁

闭包有权访问另一个函数作用域中变量的函数，即一个作用域可以访问另一个函数
闭包的作用：延伸变量的作用范围

function fn1() {
  var num = 99;
  function fn2() {
    console.log('num', num);
  }
  fn2();
}
fn1();
