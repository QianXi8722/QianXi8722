constructor 函数只要new生实例时，就会自动调用这个函数 如果不写这个constructor函数类 也会自动生成这个函数

通过class 关键字创建类 类名首字符大写
类里面有个constructor 函数 可以接受传递过来的参数 同时返回实例对象
constructor 函数只要new生实例时，就会自动调用这个函数 如果不写这个constructor函数类 也会自动生成这个函数
生成实例new 不能省略
最后注意语法规范，创建类时 类名后面不要加小括号， 生成实例时 类名后面加小括号， 构造函数不需要加function

1 创建实例和生成实例

class Test{
  constructor(param1, param2,param3) {
    this.param1 = param1;
    this.param2 = param2;
    this.param3 = param3;
  }
}
var test = new Test('李先生','23','female');
console.log(test.param1);

类中添加共有方法

类的共有属性放在constructor里面
多个函数方法之间不需要添加逗号分割


class Show{
  constructor(name, age, gender) {
    this.name = name;
    this.age = age;
    this.gender = gender;
  }
  sing(song) {
    console.log(this.name + '唱'+ song);
  }
}
var show = new Show('李先生','23','female');
show.sing('征服'); // 李先生唱征服

类继承extends 和 super关键词

extends 继承
super 调用了父类中的构造函数（传值到父类中使用父类的方法）

class Father{
  constructor(x,y) {
    this.x= x;
    this.y = y;
  }
  sum() {
    console.log(this.x + this.y);
  }
}
class Son extends Father {
  constructor(x,y) {
    super(x,y);
  }
}
var son = new Son(22,50);
son.sum(); // 72

super调用父类普通函数以及继承中属性方法查找原则

继承中的属性或者方法查找原则：就近原则

  1. 继承中，如果实例化子类输入一个方法，先看子类有没有这个方法 如果有就先执行子类的
  2. 继承中，如果子类里面没有，就去查找父类有没有这个方法，如果有就执行父类的这个方法

class Father{
    say(){
        return "我是爸爸";
    }
}
class Son extends Father{
  // 方法重写
    say(){
        console.log(super.say() + "的儿子");
        //super.say()就是调用父类中的普通函数 say()
    }
}
var son = new Son();
son.say();

super必须放到子类this之前

子类在构造函数中使用super，必须放到this前面（必须先调用父类的构造方法，再使用子类构造方法）

class Father{
  constructor(x,y){
      this.x = x;
      this.y = y;
  }
  add(){
      console.log(this.x + this.y);
  }
}
class Son extends Father{
  constructor(x,y,z) {
    // 利用super调用父类的构造函数,super必须在子类this之前调用
    super(x,y);
    this.x = x;
    this.y = y;
    this.z = z;
  }
  subtract() {
    console.log(this.x - this.y);
  }
}

var son = new Son(89,30,10)
son.add();
son.subtruct();

使用类2个注意点

在es6中类没有变量提升，所以必须先定义类 才能通过实例化对象
类里面的共有属性和方法一定要加this使用
constructor里面的this指向实例对象，方法里面的this指向这个方法的调用者
